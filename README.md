# HoloMonBlenderAddonToolkit

自作の Blender アドオンをまとめた統合アドオンです。

インストールすると色々な機能を持ったアドオンを一括でインストールできます。

機能は順次追加予定です。

## 現在の機能

[ポリゴン数の削減アドオン](https://bluebirdofoz.hatenablog.com/entry/2021/10/03/055041)

## インストール手順

1.本ページの[Code] -> [Donload ZIP]からソースコードをダウンロードします。

2.ダウンロードが完了すると、HoloMonBlenderAddonToolkit.zip ファイルが取得できます。

3.Blender を起動し、[編集] -> [プリファレンス] を選択します。

4.Blenderプリファレンスが開きます。[アドオン]タブを開き、[インストール]ボタンをクリックします。

5.HoloMonBlenderAddonToolkit.zip ファイルを選択して[アドオンをインストール]をクリックします。

6.アドオンの一覧に HoloMonBlenderAddonToolkit が表示されるのでチェックを入れてアドオンを有効にします。
